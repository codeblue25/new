<h2>문제</h2>

전화번호부에 적힌 전화번호 중, 한 번호가 다른 번호의 접두어인 경우가 있는지 확인하려 합니다.<br />
전화번호가 다음과 같을 경우, 구조대 전화번호는 영석이의 전화번호의 접두사입니다.<br />
- 구조대 : 119
- 박준영 : 97 674 223
- 지영석 : 11 9552 4421
전화번호부에 적힌 전화번호를 담은 배열 phone_book 이 solution 함수의 매개변수로 주어질 때, 어떤 번호가 다른 번호의 접두어인 경우가 있으면 false를 그렇지 않으면 true를 return 하도록 solution 함수를 작성해주세요.

<h2>제한사항</h2>

- phone_book의 길이는 1 이상 1,000,000 이하입니다.
- 각 전화번호의 길이는 1 이상 20 이하입니다.
- 같은 전화번호가 중복해서 들어있지 않습니다.

<h2>🔹나의 풀이</h2>

<h3>첫번째 풀이</h3>

```python
def solution(phone_book):
    flag = True
    head = phone_book[0]
    for i in phone_book[1:]:
        if head in i:
            flag = False
    return flag
```

결과: 정확성 50.0 + 효율성 4.2 = 54.2 **실패**<br/>
for문에서 O(N) * `in()`함수에서 O(N) = O(N**2)라서 실패가 뜬 것 같다.

<h3>두번째 풀이</h3>

```python
def solution(phone_book):
    phone_book.sort()
    for i in range(len(phone_book)-1):
        leng = len(phone_book[i])
        if phone_book[i] == phone_book[i+1][:leng]:
            return False
    return True
```

결과: 정확성 83.3 + 효율성 16.7 = 100.0 **성공**<br/>


<h3>🔶다른 사람의 풀이</h3>

```python
def solution(phoneBook):
    phoneBook = sorted(phoneBook)

    for p1, p2 in zip(phoneBook, phoneBook[1:]):
        if p2.startswith(p1):
            return False
    return True
```

<h3>알게 된 것</h3>

- `startswith()` : 문자열이 특정 문자로 시작하는지를 판단하는 함수
  - True or False를 반환한다.
  - 매개변수로 특정 문자와 시작하는 위치를 정할 수 있다.
  - ```python
       message = 'Hello world Do not give up'

       print(message.startswith('Hello'))    # 'Hello'로 시작하는 문자열인지 판단
       print(message.startswith('world', 6)) # 6번 인덱스가 'world'로 시작하는 문자열인지 판단

       # 결과:
       # True
       # True
    ```
